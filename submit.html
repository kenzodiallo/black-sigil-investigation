How to Contribute

You have two options for contributing information or documents:

1. GitHub (Recommended)

- Fork the repository on GitHub
- Add your file or post according to CONTRIBUTING.md
- Submit a Pull Request

2. Secure Email (For Non-GitHub Users)

If you are not comfortable using GitHub, you may submit information via secure email:

  • Email: ....
  • Please use PGP encryption if possible
  • Include a brief description, source(s), and date

Important Notes

  • All submissions are reviewed manually.
  • Nothing is published automatically.
  • Submissions violating the rules in CONTRIBUTING.md will be rejected.
